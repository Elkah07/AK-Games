- multiplayer.js
- database.rules.json

Installation :
1. Remplace les quatre fichiers.
2. Dans VS Code, clique sur « Valider et envoyer (push) ».
3. Attends la coche verte dans GitHub Actions.
4. Ferme complètement la PWA sur les téléphones puis rouvre-la.
