FONCTIONNALITÉS
- Mode un téléphone avec passage de l’écran et réponses privées lorsque nécessaire.
- Mode multijoueur synchronisé sur tous les téléphones.
- Scores, historique de soirée, replay, jeu aléatoire et retour au salon persistant.
- Chronomètres synchronisés pour Mime, Imitation, Plaide ta cause et La Bombe.
- Votes secrets pour les quiz, scénarios et jeux de connaissance.
- Pack adulte protégé par l’option dédiée.
- Rappels de consentement, droit de passer et consommation responsable.
- Catalogue et accueil actualisés avec les 30 jeux complets.
- Cache PWA passé en V0.14.

INSTALLATION
1. Extraire le ZIP.
2. Remplacer app.js, multiplayer.js, styles.css et service-worker.js à la racine.
3. Ajouter les 18 JSON dans le dossier data.
4. Dans VS Code, cliquer sur « Valider et envoyer (push) ».
5. Attendre la coche verte dans GitHub Actions.
6. Fermer complètement la PWA sur téléphone, puis la rouvrir.

Aucune commande Firebase n’est nécessaire grâce au déploiement automatique.
Aucune règle Database n’a été modifiée.
Les personnages officiels restent en pause jusqu’au mot-clé prévu.
La prochaine grande étape est l’audit V1.0 de tous les jeux et de tous les parcours.
