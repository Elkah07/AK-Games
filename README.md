# AK'Games V0.1

Première base navigable de l'application AK'Games.

## Inclus

- Accueil
- Création d'une partie
- Mode un seul téléphone
- Interface prévue pour le multijoueur
- Options contenu adulte / mode alcool
- Création de joueurs
- Choix parmi 16 avatars temporaires
- Salon local
- Catégories de jeux
- Catalogue complet défini à ce stade
- Blind Test visible mais grisé
- Fiches placeholder pour les jeux à intégrer

## Lancer

Ouvrir `index.html` dans un navigateur.

Pour un hébergement GitHub Pages, déposer les fichiers à la racine du dépôt.

## Suite recommandée

1. Intégrer entièrement `Qui de nous ?`
2. Intégrer `Le premier qui rit a perdu`
3. Intégrer `Qui ment le mieux ?`
4. Ajouter une vraie couche multijoueur avec création/jonction de rooms
5. Remplacer les avatars emoji par les personnages officiels AK'Games
